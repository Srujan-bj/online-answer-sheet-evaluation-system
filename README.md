# 📄 Online Answer Sheet Evaluation System

An AI-powered OCR-based web application that automates answer sheet evaluation and grading.

---

## 🚀 Features

* 📸 Upload handwritten answer sheets
* 🔍 OCR extraction using EasyOCR
* 🧠 Automated evaluation system
* 📊 Real-time dashboard with marks & status
* 😊 Emotion detection using DeepFace (webcam monitoring)

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS (Tailwind)
* Backend: Flask (Python)
* OCR: EasyOCR
* Computer Vision: OpenCV
* AI: DeepFace

---

## ⚙️ How to Run

```bash
git clone https://github.com/Srujan-bj/online-answer-sheet-evaluation-system.git
cd online-answer-sheet-evaluation-system
pip install -r requirements.txt
python paper.py
```

Open `loginpap.html` in browser.
